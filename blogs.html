Difference between Var, Let and Const ?

Var is functional scope. Var can be updated and redeclared into the scope. Let The scope of a let variable is block scope. Let It can't be updated and re-decalred into the scope. Const The scope of const variable is block scope. Const can't be updated or redeclared.

Difference Arrow Function Regular function

Unlike regular functions, arrow functions do not have their own this . The value of this inside an arrow function remains the same throughout the lifecycle of the function and is always bound to the value of this in the closest non-arrow parent function.Arrow functions do not have an arguments binding. However, they have access to the arguments object of the closest non-arrow parent function. Named and rest parameters are heavily relied upon to capture the arguments passed to arrow functions.
